# Database Resources API Reference

This section contains detailed API reference documentation for all database resources available in Stacktape.

## Available Resources

- [Relational Database](./api-reference/relational-database) - Fully managed SQL databases (PostgreSQL, MySQL, etc.)
- [DynamoDB Table](./api-reference/dynamo-db-table) - NoSQL database table
- [Redis Cluster](./api-reference/redis-cluster) - In-memory data structure store
- [OpenSearch Domain](./api-reference/open-search-domain) - Search and analytics engine

Each resource page contains comprehensive configuration examples, property descriptions, and union type alternatives.
