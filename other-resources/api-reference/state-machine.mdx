This example shows a basic state machine configuration.

State machine resource

- State machines allow you to orchestrate your compute resources.

# Basic example

```yml
resources:
  myStateMachine:
    type: state-machine
    properties:
      #
      # - Type: object
      # - Required: true
      definition:
        #
        # - Type: string
        # - Required: true
        StartAt: example-value
        #
        # - Type: object
        # - Required: true
        #
        # - Type: string
        # - Required: false
        Comment: example-value
        #
        # - Type: string
        # - Required: false
        Version: example-value
        #
        # - Type: number
        # - Required: false
        TimeoutSeconds: 300
```
