# Overview

- Kubernetes clusters allow you to deploy your workloads to a fully managed AWS EKS cluster.
- AWS EKS automatically manages the availability and scalability of the Kubernetes control plane nodes responsible
  for scheduling containers, managing application availability, storing cluster data, and other key tasks.
- Stacktape makes it easy to create and manage the cluster, and configure all of the necessary AWS resources required by the EKS cluster.
- Stacktape also simplifies the deployment of your applications into that cluster.
