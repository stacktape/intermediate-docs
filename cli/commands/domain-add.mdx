# Overview and basic usage

Adds specified domain to your account.

<br />

- Added domain and its subdomains can then be used in your stacks (Web Services, Hosting Buckets, HTTP Api Gateways,
  Application Load Balancers and CDNs).
- Before adding a domain, carefully read
  [Domains and certificates](https://docs.stacktape.com/other-resources/domains-and-certificates/#adding-domain) page.

{/* WRITE ONLY BELOW THIS LINE */}

{/* WRITE ONLY ABOVE THIS LINE */}

```bash
stacktape domain:add --region <<region>>
```
