# Overview and basic usage

Configures your Stacktape API Key for the current system

<br />

- All of the subsequent stack operations will be attributed to the user and organization having this API Key.
- You can get your API key at https://console.stacktape.com/api-keys.
- You can specify the API key using the `--apiKey` option or interactively using CLI.

{/* WRITE ONLY BELOW THIS LINE */}

{/* WRITE ONLY ABOVE THIS LINE */}

```bash
stacktape login
```
