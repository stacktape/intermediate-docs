# Overview and basic usage

Compiles CloudFormation template and saves it into the specified file.

<br />

- By default, the template is saved to `./compiled-template.yaml`. To override this, use the `--outFile` option.

{/* WRITE ONLY BELOW THIS LINE */}

{/* WRITE ONLY ABOVE THIS LINE */}

```bash
stacktape compile-template --stage <<stage>> --region <<region>>
```
